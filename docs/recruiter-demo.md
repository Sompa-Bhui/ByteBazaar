# Recruiter Demo Script

1. Open the ByteBazaar homepage and point out that it is a developer-focused marketplace, not a generic storefront.
2. Click **Open Build Lab** and explain that devices become a graph with typed compatibility rules.
3. Load **MacBook Dual Display Problem**.
4. Show the solver output, conflict root cause, and ranked fix preview.
5. Apply the fix transaction and point to the before/after proof.
6. Undo the transaction to show exact restoration.
7. Load **Underpowered Dock**.
8. Show the power delivery conflict and recovery option.
9. Apply the recovery through the same transaction engine.
10. Open **FUTURE** mode with **Cheap Now, Expensive Later**.
11. Compare Cheapest vs Balanced vs Future-proof and explain replacement waste.
12. Open **SIMULATE** mode and show blast radius and critical components.
13. Finish by summarizing the architecture: graph, solver, simulator, regret, transaction engine, revision history, and deterministic replay.
